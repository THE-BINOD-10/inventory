Follow the below instructions to grant API access to RetailOne:
  • Go to http://developer.amazonservices.in

  • Enter Application name and description as 'RetailOne' (or whatever you prefer to identify later)
  • In the newly generated table row copy Application Id and Application Secret and paste it in the corresponding fields above
